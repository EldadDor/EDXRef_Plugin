Validates @WSHeader annotations defined within the @WSHeaders container on consumer types.

This inspection checks classes and interfaces that are annotated with @WSConsumer and implement/extend WebserviceConsumer or PearlWebserviceConsumer.

It reports the following issue:

  • Error: Missing Default Value: If a @WSHeader defined inside @WSHeaders on the type does not have a defaultValue attribute, or if the value is empty (""), an error is reported.

Example:


@WSConsumer(url = "...")
@WSHeaders({
    @WSHeader(name = "X-Request-ID", defaultValue = "ID123"), // OK
    @WSHeader(name = "Accept"), // ERROR: Missing defaultValue
    @WSHeader(name = "Content-Type", defaultValue = "") // ERROR: Empty defaultValue
})
public interface MyConsumer extends WebserviceConsumer { }
